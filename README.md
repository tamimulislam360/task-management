

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tech Stack

- Next.js
- Tailwind
- MST (MobX State Tree)


## Functionality


- CRUD (Read, Create, Update, Delete) Tasks
- Data stored in local storage
